将Hexo的工程文件同步到Github

上传

  1. 在hexo目录下执行git init在本地建立一个git仓库

  2. 建立.gitignore文件，内容如下

    (1) 【存疑】我的butterfly主题安装在node_modules目录下，我想将它上传，但这样配置的.gitignore文件不起作用。

    (2) 于是我手动强行添加的hexo-theme-butterfly文件夹到git仓库中

  3. 在github上建立一个新的仓库，最好是私有的。并链接到本地参数。

  4. 然后按照正常的方式将本地仓库pull到github中

下载

  1. 首次下载需要git clone，ssh的方法

  2. 然后由于clone下来的文件夹不包含node_moduls文件夹的内容，需要在目录下执行npm install，安装hexo需要的依赖。此时就可以在新的路径下编辑hexo了。
